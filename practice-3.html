お題を２０分くらいで書く練習

    
      次の条件を満たすユーザーの name だけを配列で返す 関数を作ってください。

      const users = [
        { id: 1, name: "Taro", score: 82, isActive: true },
        { id: 2, name: "Hanako", score: 65, isActive: true },
        { id: 3, name: "Ken", score: 90, isActive: false },
        { id: 4, name: "Yuki", score: 75, isActive: true },
        { id: 5, name: "Sato", score: 50, isActive: false }
      ]

      ▼条件
      isActive が true
      score が 70以上
      score が高い順に並べる

      ▼考え方
      isActive が true の user を抜き出す
      スコアが70以上の user を抜き出す
      スコアが高い順にソートする
      結果からnameを抜き出し、配列に入れる
      配列を返す
